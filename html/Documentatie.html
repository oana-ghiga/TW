1. Abstract

The Digital Herbarium is an online collection of documented plant specimens for scientific research and education. The Digital Herbarium Website brings this collection online and accessible to anyone worldwide. Users can learn about plant characteristics, habitats, uses, and discover new species.

2. Introduction

2.1 Purpose

The purpose of the Digital Herbarium Website is to provide an online platform for learning about plants and their attributes. The website aims to make a vast collection of plant specimens easily accessible to users from anywhere in the world, along with their detailed information and photographs. It aims to support scientific research, educational purposes, and general interest in botany and plant sciences. By providing a rich collection of plant species, users can learn about their characteristics, habitats, and uses. Moreover, the platform may also allow users to contribute their observations and comments on plants, thereby expanding the knowledge and understanding of the plant world. In summary, the purpose of the Digital Herbarium Website is to facilitate learning and promote awareness of the importance of plants in our ecosystem.

3. Overall Description

3.1 Product Perspective

The Digital Herbarium Website is a university project designed to showcase the potential of online plant specimen collections for research and educational purposes. The website will be developed as a standalone application, primarily for demonstration purposes, and will not be integrated with any other software systems.

3.2 Product Functions

The website offers a range of functions and features that enable users to interact with the plant collection in various ways. Users can create personalized albums for plants, save plants in those albums, and add comments for plants. Additionally, users can create an account and log in to access their albums and comments from any device. The home page of the web app is composed of a variety of plant photos that users can click to reveal detailed descriptions and comment sections for each plant. The system also generates thematic albums for users, such as medicinal plants, mountain plants, and others. Users can search plants and filter their main page feed based on different categories, such as medicinal plants, mountain plants, and other themes. Overall, the system is designed to facilitate the exploration and learning of plant attributes and characteristics for users with different levels of expertise and interests.

3.3 User Classes and Characteristics

The Digital Herbarium Website is designed to accommodate a variety of user classes with different levels of expertise and interests in plants. The primary user classes are students, researchers, and plant enthusiasts. 3.4 Operating Environment

The Digital Herbarium Website will be a web-based application that can be accessed from any modern web browser on a desktop, laptop, tablet, or mobile device. The application will be designed to be compatible with commonly used web technologies, such as HTML5, CSS, JavaScript and NodeJS.

3.5 Operating Environment

The following constraints apply to the design and implementation of the Digital Herbarium Website:

  • The website must be designed and implemented using HTML, CSS, JavaScript, NodeJS and MySQL Database technology
  • The website must be compatible with modern web browsers such as Google Chrome, Mozilla Firefox, and Safari,
  • The website must be designed to be responsive and adaptable to different screen sizes, including desktop and mobile devices,

3.6 User Documentation

The user will have a user guide on how to use the web application.

3.7 Assumptions and Dependencies

  • The web application assumes that users have a basic understanding of how to use a web browser and have access to an internet connection,
  • The application needs a browser that supports HTML5, CSS, JavaScript and NodeJS.

5. System Features

5.1 Registration Form

The system shall provide a registration form for users to create an account.

5.2 Filtered Search

The system shall allow users to search and filter the main page feed based on various criteria such as plant name, category, and tags.

5.3 Album Recommendation

The system shall generate and recommend thematic albums to users based on the user's plant collection and search history.

5.4 Album Creation

The system shall allow users to create custom albums for their plant collections.

5.5 Adding Plant to Album

The system shall allow users to add plants to their custom albums.

5.6 Add Plant Comment

The system shall allow users to add comments for plants and view comments from other users.

5.7 Top Most Collected Plants

The system shall provide a feature to display the most collected plants by users in a separate section of the main page feed.

6. Other Nonfunctional Requirements

6.1 Performance Requirements

The web application must be able to handle a large number of simultaneous user requests, with minimal response time for searching, browsing, and displaying plant information.

6.2 Safety Requirements

The app must provide accurate information on plants to ensure the safe use of plants and prevent any harm to users.

6.3 Security Requirements

The system must provide secure user authentication and session management. Passwords must be encrypted and stored securely.

6.4 Software Quality Attributes

The app should have robust security measures in place to protect user data and prevent unauthorized access. The app should have fast response times, to ensure that users can quickly access the information they need. The app should have a user-friendly interface, with clear and intuitive navigation, to ensure ease of use for all users.

6.5 Business Rules

The app must not promote or endorse the consumption or use of illegal or harmful plants or products. The app must provide reliable and up-to-date information about plants and their characteristics, uses, and risks. Users must register and log in to create and manage their albums, add comments, and access personalized recommendations.

7. Other Requirements

A MySQL database used to store user account details, plants, saved albums and comments.
